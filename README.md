# BIRDS - Bird Identification via Real-time Detection System

This repository is from the **BIRDS** project.

The model in this repo was trained to classify **bird vs not bird** using this notebook:
https://colab.research.google.com/drive/18Sh_G6UzMayeEFLGDky-rUCnaivUEBNi?usp=sharing

## Dataset setup for training in Google Colab

To run the same training:

1. Download the dataset:
   https://drive.google.com/file/d/1Ok65EMsQvS6qG65C6dWZ7j3R97C-R0fN/view?usp=drive_link
2. Add the dataset to your Google Drive, in **MyDrive**.
3. The dataset can also be placed in a different folder, but then you must update the dataset path in the notebook.
